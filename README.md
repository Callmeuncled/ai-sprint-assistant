# 🤖 AI Sprint Assistant

KI-gestützter Sprint-Assistent gebaut mit n8n + Groq (Llama 3)

## Was es macht
- Nimmt Sprint-Tasks entgegen
- KI analysiert und fasst zusammen
- Erstellt automatisch Statusberichte auf Deutsch

## Technologien
- n8n (Workflow Automatisierung)
- Groq API (kostenloser LLM)
- Docker
